# **DEMOQA Automation**

This is the automation for the website https://demoqa.com/books

### Tech Stack used:
1. Selenium	
2. TestNG
3. POM
4. Maven


### Requirements:
* Java 11 and above

### Installation & Setup:
* Clone this project
* Open using Intellij IDE
* Maven will install the dependencies automatically

### To Run the test:

`mvn clean test`
